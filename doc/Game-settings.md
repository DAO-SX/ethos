## Settings
+ Rules，规则，属于更底层的通信设定，保障元宇宙顺畅、安全、高效运转为目标。
+ World setting，属于全局级别设定，元宇宙建立时已经设定OK，不可调整。
+ World setting 设计较少，是为了建立不同外部和内如特征的元宇宙进行的基础设定。
+ God setting，属于Lord进行的设定，一般属于一个社区来购买后进行设定，属于元宇宙内部的大的设定。
+ 以下以我们的阶段MVP： 兽人小世界为例。
